Operators
	Comparison: <, >, <=, >=, ==, !=
	Logical: and, or, not
	Arithmetic: +, -, *, /, **, %
	Augmented Assignment: +=, -=, *=, /=, **=, %=

import - Add modules to Python
	System libraries
	User libraries

	Syntax:
		import <module>
		from <module> import <object>
		import <module> as <name>
		from <module> import <object> as <name>
	
	can use dir() to examine a module's contents
